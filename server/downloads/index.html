Installation Guide

Windows
macOS
Android
1
Download the .exe installer using the button above.
2
Run the installer. If Windows SmartScreen appears, click "More info" then "Run anyway".
3
Choose your installation directory (or keep the default) and click Install.
4
Launch RMPG Flex from the desktop shortcut or Start Menu. The app connects to the server automatically.
Windows SmartScreen note: Since the installer isn't signed with an EV certificate, Windows may show a warning on first run. Click "More info" and then "Run anyway" to proceed. The app is safe to install.
1
Download the .dmg file using the button above.
2
Open the downloaded .dmg file. Drag RMPG Flex into the Applications folder.
3
Important — Remove quarantine flag: Open Terminal (search for it in Spotlight) and run this command:
sudo xattr -cr /Applications/RMPG\ Flex.app
Enter your Mac password when prompted. This clears the macOS quarantine flag.
4
Now open RMPG Flex from the Applications folder. If macOS still asks, right-click the app and select "Open", then click "Open" in the dialog.
5
The app will start and connect to the server automatically. Future launches work normally from the Dock or Launchpad.
Getting "damaged and can't be opened"? This is normal for apps distributed outside the App Store. Open Terminal and run: sudo xattr -cr /Applications/RMPG\ Flex.app — this removes the macOS quarantine flag. After that, the app will open normally.
1
Download the .apk file using the button above, or scan the QR code.
2
Open the downloaded APK. If prompted, go to Settings > Security and enable "Install from Unknown Sources" for your browser.
3
Tap Install when prompted. The app will install on your phone.
4
Open RMPG Flex from your app drawer. Sign in with your credentials to access the dispatch system.
Android security note: Since this app is not from the Google Play Store, your phone will ask you to allow installation from this source. This is normal for internally distributed apps. You can disable the permission after installation.
